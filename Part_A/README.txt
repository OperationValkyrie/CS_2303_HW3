Assignment 3 - String Functions
README

This program simulates the actual C-style string functions in the standard libaries. These functions include strdup(), strlen(), strcpy(), trcat(), strncat(), strncpy(), and strndup(). As part of the standard libaries the created functions do have any and all fault of the standard libaries including the ability to not work propertyly if given invalid input.

This program can be complied with the following command
make ctest

This program can then be run with the following command
./ctest

This program requires no arguments

In Eclipse, this program can be complied by right-clicking on the project name and selecting build project

In Eclipse, this program can be run be left-clicking the run command - the run configuration requries no arguments
