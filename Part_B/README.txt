README

This program demonstrates the use of a struct know as an employee. Employees can be created through specific parameters or can be randomly generated. Furthermore, lists of employees can be randomly generated and list can be copied both shallowly and deeply. Finally, lists may be freed up.

This program can be complied with the following command
make structest

This program can then be run with the following command
./structest

This program requires no arguments

In Eclipse, this program can be complied by right-clicking on the project name and selecting build project

In Eclipse, this program can be run be left-clicking the run command - the run configuration requries no arguments
